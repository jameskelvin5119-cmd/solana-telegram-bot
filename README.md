# Solana Telegram Bot

A Telegram bot that tracks Solana wallet activity in real time using Helius webhooks.

## Features
- Track Solana wallet transactions
- Real-time alerts via Telegram
- Powered by Helius enhanced webhooks
- Lightweight Python backend

## Tech Stack
- Python
- Telegram Bot API
- Helius API (Solana)
- Webhooks
